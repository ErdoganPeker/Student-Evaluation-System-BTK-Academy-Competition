---

## 🏆 Yarışma Başarısı

Bu proje **BTK Akademi** tarafından düzenlenen öğrenci performans değerlendirme yarışmasında **100. sıraya** ulaşmıştır. Yüzlerce katılımcı arasından elde edilen bu derece, uygulanan ML/DL yaklaşımının etkinliğini doğrulamaktadır.

---

## ✨ Özellikler

- BTK Akademi yarışma veri seti üzerinde kapsamlı EDA
- Öğrenci başarısını etkileyen faktörlerin istatistiksel analizi
- Çoklu makine öğrenmesi modellerinin karşılaştırmalı değerlendirmesi
- Derin öğrenme modeli entegrasyonu
- Feature engineering ve veri ön işleme pipeline'ı
- Çapraz doğrulama (cross-validation) ile güvenilir model ölçümü

---

## 📊 Analiz Akışı

```
Veri Yükleme → Keşifsel Analiz (EDA) → Veri Temizleme
      ↓
Feature Engineering → Model Eğitimi → Hiperparametre Ayarı
      ↓
Model Karşılaştırma → En İyi Modeli Seçme → Tahmin → 100. Sıra
```

**İncelenen Başarı Faktörleri:**
- Demografik özellikler (yaş, cinsiyet, bölge)
- Ders katılımı ve çalışma alışkanlıkları
- Önceki akademik performans
- Sosyoekonomik göstergeler
